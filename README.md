##   FUNCTIONS :
#### Buttons  :
	-  Step on/off
	-  Shift + step button = step jump
	-  Fill
	-  Fill + shift = Clear
	-  Copy/Paste
	-  Copy/Paste + shift = Save
	-  Pause/Play
	-  Pause/Play + shift = Stop
	    

#### Encoders :
	- Tempo
	- Tempo + push = clock divider
	- Gate length
	- Gate length + push = Number of step
	- Page navigation
	- Page navigation + push = Play mode
	  
#### Selecter :
	- Part choice

#### Play mode :
	- Forward
	- Backward
	- Ping Pong
	- Random
	- Brownien
